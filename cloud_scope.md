# Cloud Scope

## Scenario
A cloud-hosted financial services application supporting customer authentication, account access, and transactions.

## In Scope
Identity and access management
Network exposure and segmentation
Data storage security
Logging and monitoring
Secrets management
Baseline configuration and hardening
Backup and recovery posture

## Out of Scope
Application source code review
Penetration testing
Physical security
Mobile application threats

## Objective
Identify common cloud misconfigurations, quantify risk, and recommend controls aligned to NIST CSF and CIS Controls.
