## What This Project Demonstrates
- Identification of common cloud security misconfigurations
- Risk prioritization using likelihood and impact scoring
- Mapping of risks to preventive and detective controls
- Alignment with NIST CSF and CIS Critical Security Controls
- Executive-level cloud risk communication
